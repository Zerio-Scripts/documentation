# Description

A sleek and easy to understand experience for your players when buying vehicles. Why make it hard when it can be easy?

## Requirements

- SQL resource: oxmysql, ghmattimysql, mysql-async
- Framework: QBCore, ESX

## Supported by default

- Job/Society money: esx_society, okokBanking, qb-management, qb-banking, qb-bossmenu, renewedbanking, 710-management, fd_banking
- Target script: ox_target, qtarget

<Callout>
  Support for any job/society script can be added via the
  `zerio-cardealer/server/functions.lua` file.
</Callout>
