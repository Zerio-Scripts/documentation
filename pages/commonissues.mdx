## üö® Emergency Troubleshooting

### Most Common Issues (Quick Fixes)

**Resource Won't Start:**

1. Check if all dependencies are installed and started
2. Verify the resource is properly named (no spaces, special characters)
3. Ensure SQL files have been imported
4. Check console for specific error messages

**"Lacking Entitlement" Error:**

1. Restart your FiveM server completely
2. Verify your license key ownership matches your purchase account
3. Check if the resource is properly downloaded from Keymaster

**Players Can't Access Features:**

1. Verify framework compatibility (QBCore vs ESX)
2. Check job permissions and grades
3. Ensure target system is configured correctly
4. Test with admin permissions first

**Database Errors:**

1. Confirm SQL files have been imported
2. Check database connection configuration
3. Verify table names match the resource expectations
4. Test database connectivity

## üìã Information to Gather Before Seeking Help

When reporting issues, please provide the following information to help us assist you more effectively:

### Required Information

1. **Resource Name**: Which Zerio-Scripts product is causing issues
2. **Server Framework**: QBCore, ESX, or standalone setup
3. **FiveM Version**: Your server's FiveM version number
4. **Error Messages**: Complete console output including timestamps
5. **Configuration**: Relevant sections of your config files

### Helpful Additional Information

- **Server Specifications**: RAM, CPU, operating system
- **Other Resources**: List of other resources that might conflict
- **Recent Changes**: Any recent server modifications or updates
- **Reproduction Steps**: Detailed steps to reproduce the issue
- **Screenshots/Videos**: Visual evidence of the problem

### Console Log Example

When sharing console logs, include relevant context:

```
[2024-01-15 10:30:45] [INFO] Starting resource: zerio-bossmenu
[2024-01-15 10:30:45] [ERROR] zerio-bossmenu: Database connection failed
[2024-01-15 10:30:45] [ERROR] zerio-bossmenu: Please check your database configuration
[2024-01-15 10:30:45] [WARN] zerio-bossmenu: Resource failed to start properly
```

### Support Guidelines

**Before Creating a Support Ticket:**

1. Check this troubleshooting guide thoroughly
2. Search our Discord for similar issues
3. Test with minimal configuration/other resources disabled
4. Document the issue with screenshots or videos

**When Creating a Support Request:**

- **Be Specific**: Provide exact error messages and steps
- **Be Patient**: Our team responds as quickly as possible
- **Be Collaborative**: We may ask for additional information
- **Be Respectful**: Help us maintain a positive community

- **Feature Requests**: 1-3 business days
